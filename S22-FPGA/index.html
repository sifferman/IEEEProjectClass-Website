What is this class?

This class will cover how to use an FPGA to produce VGA output. We will be using the Tang Nano with open source tools, similar to the project completed here: FPGA Screensaver Project.

By completing this project, you will learn best practices for writing Verilog, which will be crucial if you plan on pursuing digital design.
